一 HC-0X 藍芽模組簡介
１ 伺服馬達的構造
　　目前的3C 產品相當普遍的出現在我們生活周遭，尤其手持式裝置的手機和平板，更是如影隨形的跟著我們生活移動，因此若能結合手機的藍芽裝置來與電子產品或藝術作品產生互動，將可以使我們的生活更加便利，讓藝術作品更加賦予價值與生命。因此本單元將教您認識與使用藍芽模組，並結合市面上手機APP 軟體來與藍芽模組產生互動，也就是利用手機的APP 軟體來與藍芽結合，做成各種控制應用，可增加產品的實用性與趣味性。　　市面上有很多款式的藍芽接收模組，在此我們將介紹最普遍常見的HC-05及HC-06 這兩款的藍芽模組，HC-05 和HC-06 的硬體相同，都採用英國劍橋的CSR（Cambridge Silicon Radio）公司的BC417143 晶片，支援藍牙2.0+EDR 規範，只是晶片內部的韌體不同而已。
