一、Arduino的介紹
　Arduino 開發團隊中的David Cuartielles：「現在因為有標準化以及專利這些系統，讓許多人被剝奪了去學習事物和技巧，也許只有少數的一些人譬如駭客才能去深入了解。我認為開源硬體最大的價值就是讓其他人也可以用合法、合乎道德的方式去理解事物運作的原理，讓人不只是停留在修理物品的層次，而是理解生活周遭事物運作的原理，我認為對當代社會是非常重要的。」
　　Massimo Banzi 說：「因為開源硬體使得人們更容易地開發自己的產品。人人都能用Arduino 開發出驚奇的創意。」
　　在Arduino 的世界裡，人人都可以是自造者、產品的設計師，即使你毫無電學電路、軟體背景，在開源社區裡的開放資源，可以讓你藉由社群裡他人分享出的珍貴經驗，快速的創造出屬於你自己的獨特創意。
